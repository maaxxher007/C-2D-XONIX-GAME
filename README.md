# Xonix Game (SFML)

A C++ arcade-style territory capture game inspired by Xonix, built with SFML.

## Developer
- Name: Maaz Sher
- Email: shermaaz55@gmail.com
- Role: AI/ML Developer

## Project Overview
This game includes:
- Main menu navigation
- Single-player and multiplayer game modes
- Difficulty selection (Easy, Medium, Hard, Continuous)
- Enemy movement and area claiming mechanics
- High score persistence
- Background music and game assets

## Main Files
- `main.cpp`: Core gameplay loop, menus, rendering, scoring, and game logic.
- `outback.cpp`: Reserved/auxiliary source file.
- `flowchart.drawio`: Game flowchart for logic visualization.

## Assets Used
- Images: `images/`, `image.jpg`, and multiple mode-background images
- Audio: `music.wav` / `music.mp3`
- Font: `Arimo-Bold.ttf`

## Build Notes
This project is a C++ SFML game. Ensure SFML is installed and linked correctly for your compiler.

Example (conceptual):
1. Compile `main.cpp` with SFML graphics, window, system, and audio libraries.
2. Run the executable from the project root so relative asset paths resolve correctly.
